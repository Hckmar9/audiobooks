<h1 align="center">PyPDF library to read PDF books 🤖</h1>

> A basic syntax to read PDF books in case of laziness.

## What you need?
- [pyttsx3](https://pypi.org/project/pyttsx3/)
- [PyPDF2](https://pypi.org/project/PyPDF2/)

## Developed with PyCharm 🐍
- pip install pyttsx3
- pip install PyPDF2
